LNMP环境搭建（一）虚拟机安装精简centos7.2

准备工作

  1. 虚拟机：个人使用是旧版的vmwareworkstation10，虚拟机根据自己习惯选择;
  2. 镜像文件：下载 centos7.2版本，CentOS-7-x86_64-DVD-1511.iso;

开始安装 VM虚拟机

  1. 为了避免丢失重要数据，建议找一个比较空闲的分区，新建文件夹centos7.2;

  2. 打开虚拟机，新建虚拟机，典型 -> 选择新建一个空白硬盘 -> 选择Linux系统，版本选择Centos 64位 -> 安装位置选择刚刚新建的文件夹 centos7.2 -> 磁盘大小默认20G即可，选择将虚拟磁盘存储为单个文件

  3. 自定义硬件：

    内存：内存默认1G,但在搭建LNMP环境的时候，为了提高编译安装的速度，可以根据自己物理机内存多划分点给虚拟机，8G物理机可以划分3G左右。

处理器；1核即可；

CD/DVD：勾选启动时连接，勾选使用ISO镜像，选择刚刚下载的镜像。

网络适配器：勾选启动时连接，勾选自定义特定虚拟网络，勾选VMnet1(NAT模式);

USB控制器：默认即可;

声卡：（声卡用不到，点击下面的移除）;

打印机：（打印机用不到，点击下面的移除）;

显示器：默认即可;

  4. 点击 完成，空白虚拟机创建完成。
